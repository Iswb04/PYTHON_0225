1- diagrama DER - Diagrama Entidade Relacionamento
    --> Entidades: Retangulos (tabelas (ex: users))
    --> Atributos: Colunas (ex: id, first_name, last_name, email)
    --> PK: Chave primaria que nunca se repete
    --> FK: Chave estrangeira
    --> Relacionamento 1:1 | 1:N | N:N

* instalar o docker
    --> Docker é uma plataforma usada para criar, executar /
    e gerenciar containers que são ambientes isolados /
    (como mini máquinas virtuais, mas muito mais leves).

    > principais usos para o docker:
      1 - Padronizar ambientes (todos os devs usam o mesmo ambiente)
      2 - Facilitar o deploy (empacota a aplicação em uma imagem Docker)
      3 - Isolamento (cada container é independente)
      4 - Escalabilidade (executa containers múltiplos de forma simples)
      5 - Integração com CI/CD (como Github)

    --> No Windows, o Docker (rodando via WSL2) /
     não aceita caminhos com barra invertida! 
     * Forma correta: /c/Users/...
