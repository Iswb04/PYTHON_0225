6 - pip install django

    -> django-admin / deactivate
    -> django-admin startproject project .   (. joga na raiz da pasta)
  * -> manage.py = comando central do django
        -> python manage.py runserver (comando para subir um servidor)
        -> python manage.py startapp nome_do_app (criar user admin)
        -> O QUE É: framework web em Python usado para criar sites /
        e sistemas web completos de forma rápida e organizada.


    | Coisa     | Flask           | Django          |
    | --------- | --------------- | --------------- |
    | Login     | Você cria tudo  | Já vem pronto   |
    | Banco     | SQL manual      | *ORM automático |
    | Segurança | Você implementa | Vem integrado   |

    | Estrutura | Livre           | Padronizada     |

* ORM é o que permite você trabalhar com banco de dados usando código /
Python, sem escrever SQL diretamente.
